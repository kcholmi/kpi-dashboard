# KPI Sales Dashboard
[Tableau Public link](https://public.tableau.com/views/InteractiveSalesDashboard_17635860507800/USSalesDashboard?:language=en-US&:sid=&:redirect=auth&:display_count=n&:origin=viz_share_link)

A simple interactive KPI dashboard made using Tableau, allowing for easy display of key KPIs, interactivity and filtering, and broad sales performance and trends to provide insight for executives.
[Preview](preview.png)


## Dataset
Fictional sales data, organized by individual item sales.

## Analysis Objectives
1. Are they any sales trends?
2. What are our top KPIs?
3. Are sales trending up or down?
4. Which states are struggling or performing well?


## Key Findings
- Top Profitable categories
- Most profitable customer segments
- Sales vs Profit per state
- Average Order Value
